# Aves de Mallorca - Una pequeña guía ornitológica

Práctica de CSS y JavaScript — Interfaz web interactiva sobre aves de la isla de Mallorca.

---
## Idea general del proyecto

El proyecto consiste en una pequeña guía general de ortnitología interactiva sobre las principales aves que se pueden visualizar en Mallorca.
Está dividida en tres secciones: 
1. **Mapa** — Un mapa SVG de la isla con 5 zonas clicables. Al hacer clic en una zona se muestra un panel lateral con las aves que habitan esa área y se pueden escuchar sus cantos.
2. **Aves** — Catálogo completo de 13 especies con tarjeta visual, descripción, nombre científico, hábitat y botón de sonido. Incluye filtros por zona.
3. **Rutas** — Tres itinerarios ornitológicos recomendados con dificultad, duración y aves destacadas.

## Principales decisiones de diseño

- **Paleta natural y suave**: cremas, verdes salvia, cielos, tierras y arenas. Ningún color estridente.
- **Tipografía**: *Playfair Display* (serif, itálica) para títulos elegantes; *Raleway* (sans-serif ligera) para textos.
